# ropper-ng
New generation Ropper with least power but more agile.

## Usage
### Help:
```
python3 ropper-ng.py --help
```

```
python3 ropper-ng.py <binary>
```

```
python3 ropper-ng.py <binary> --gadget 'pop esp; ret'
```
